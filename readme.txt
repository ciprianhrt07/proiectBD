Instalarea si configurarea aplicatiei:


Pentru MySQl Workbench

 1. rulare "creare tabele" 
 2. rulare "triggere" 
 3. rulare "inserari in tabele"
 4. rulare "view-uri"
 5. rulare "proceduri si functii" 
 6. rulare "interogari"
 7. rulare "tabela cu utilizatori"

Pentru deschiderea aplicatiei din Eclipse:
    1. creare proiect nou java
    2. copiere clase log si  windowinfo
    3. rulare program din fereastra log
    4. conectare cu utilizatorul potrivit (user:root si pass: root pentru administrator)
    5. executare diverse functii prin apasarea butoanelor
       si inserarea in textField ul de sus
    6. orice neconcordanta intre datele introduse
       in textField si argumentele necesare executarii 
       unei actualizari inserari sau modificari va fi 
       semnalata printr-o fereastra de dialog.
